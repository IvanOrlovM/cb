+ Используя Visual Studio, создайте проект "ArrayManager" по шаблону ConsoleApplication.
Требуется:
+ Создать массив размера N элементов, (10 min)
    + размер массива задает пользователь (15 min)
- заполнить массив произвольными целыми значениями.
    + создать цикл для инициализации элементов (20 min)
    - инициализировать массив случайными числами в диапозоне от 0 до 100
Вывести на экран:
+ наибольшее значение массива, (5 min)
+ наименьшее значение массива, (1 min)
+ общую сумму всех элементов,  (1 min)
+ среднее арифметическое всех элементов (1.5 min)
+ вывести все нечетные значения элементов массива. (30 min)

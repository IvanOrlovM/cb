- Используя Visual Studio, создайте проект "ArrayManager" по шаблону ConsoleApplication.
Требуется:
- Создать массив размера N элементов, 
    - размер массива задает пользователь 
- заполнить массив произвольными целыми значениями.
    - создать цикл для инициализации элементов 
    - инициализировать массив случайными числами в диапозоне от 0 до 100
Вывести на экран:
- наибольшее значение массива, 
- наименьшее значение массива, 
- общую сумму всех элементов,  
- среднее арифметическое всех элементов 
- вывести все нечетные значения элементов массива.
